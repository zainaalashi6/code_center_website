Project Name: شيفرة كود - واجهة موقع التدريب.

Open index.html in a browser.

This project uses pure HTML, CSS, and JavaScript.

The folder structure is simple and clear.

Files in assets/ include logo and sample images.

Images in portfolio can be customized by replacing files inside assets/.

The main styling is located in css/style.css.

JavaScript logic is inside js/app.js.

See js/app.js for courses array.

You can edit course names, descriptions, duration, and price.

Booking form submissions are saved in localStorage.

To view saved bookings, open courses.html and scroll down.

The site is fully responsive for mobile and desktop.

Portfolio images open inside a modal when clicked.

Contact form sends a success alert on submit.

No backend is required for this project.

You can upload this project to GitHub easily.

You can use Live Server extension in VS Code for better viewing.

This project is designed for university training purposes.

Enjoy coding! ✨
